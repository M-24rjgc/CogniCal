# Product Overview

## Product Purpose

CogniCal 是一款面向个人的先进 AI 驱动桌面日程管理应用，通过 Chain-of-Thought（CoT）推理、蒙特卡洛树搜索（MCTS）与本地分析引擎，将自然语言需求转化为结构化任务、智能排期和可执行洞察。系统基于 Tauri 2 桌面架构，所有用户数据本地存储于 SQLite 数据库确保隐私，同时通过 DeepSeek API（需要网络连接）提供强大的 AI 能力，实现原生性能与智能化体验的完美结合。

## Target Users

- **多项目知识工作者**：并行负责多个交付物，期待智能优先级判断、冲突预警与排期建议，需要高效的任务规划工具。
- **自驱型学习者与研究者**：依赖精细的学习计划、复盘仪式与自我监督来维持长期目标推进，追求持续成长。
- **独立创作者 / 自由职业者**：强调工具灵活度与数据主权，不希望被订阅制或云端锁定约束，重视隐私与自主权。
- **深度专注型工作者**：需要在高强度专注与生活节奏之间取得平衡，期望获得温和、可信赖的智能提醒机制。

## Key Features

1. **AI 驱动的自然语言任务解析**：基于 DeepSeek API 与 CoT 推理链，将自由文本智能拆解为标题、优先级、截止时间、计划开始时间、预估工时等结构化字段，并生成可解释的推理过程，极大简化任务录入流程。
2. **智能任务规划与冲突化解**：通过 MCTS 算法生成多套可执行排程方案，自动识别时间/依赖冲突并提供权衡建议，支持用户一键采纳、微调或反馈，确保计划的可行性与灵活性。
3. **深度生产力洞察分析**：本地 Rust 引擎离线分析任务完成趋势、时间分配模式、生产力评分与工作负载预测，通过 shadcn/ui 仪表盘呈现可视化洞察，帮助用户持续优化工作习惯。
4. **双智能体协同系统**：任务规划智能体与生活助手智能体分工协作，实现任务执行与生活节奏的动态平衡，守护可持续的工作方式。
5. **隐私优先桌面架构**：Tauri 2 + Rust + React 技术栈实现原生性能、完整系统集成（通知、托盘、文件系统），所有数据本地存储零云端依赖，AI 功能通过安全的 API 调用实现，在保护隐私的同时提供强大的智能化能力。

## Business Objectives

- **效率提升**：将用户的任务规划时间缩短 30%，通过 AI 自动解析和智能推荐提高工作效率。
- **执行保障**：提高任务准时完成率至 75% 以上，通过智能调度和冲突预警确保计划可执行性。
- **用户粘性**：维持每日活跃用户占比 ≥ 60%，月留存率 ≥ 50%，通过透明的 CoT 推理和持续优化建立用户信任。
- **技术领先**：打造业界领先的个人 AI 生产力工具，结合桌面应用的原生体验与先进的 AI 能力。
- **生态构建**：构建可持续的功能演进节奏，为未来的双智能体协同、团队协作与深度个性化学习功能奠定坚实基础。
- **开源共建**：保持完全开源免费，通过社区驱动发展，在零服务器成本前提下交付旗舰级用户体验。

## Success Metrics

- **规划效率**：连续 4 周使用后，任务规划时间中位数下降 ≥ 30%。
- **执行成效**：任务按时完成率提升 ≥ 15%，延期任务占比 < 10%。
- **活跃度**：7 日滚动活跃率 ≥ 60%，30 日留存率 ≥ 50%。
- **稳定性**：桌面应用崩溃率 < 0.1%，关键功能可用性 > 99.9%。
- **满意度**：净推荐值（NPS） ≥ 50，AI 建议采纳率 ≥ 65%。

## Product Principles

1. **隐私优先**：所有用户数据本地存储于 SQLite 数据库，任务内容、分析结果与个人习惯数据完全掌控在用户手中，默认不采集、不上传、不共享。
2. **AI 透明可解释**：每次 AI 推理（任务解析、规划生成、优先级建议）都展示完整的 Chain-of-Thought 推理过程，提供清晰的假设、证据与备选方案，确保用户理解并信任 AI 决策。
3. **人机协同决策**：AI 提供智能建议但用户始终掌控最终决策，支持接受、拒绝、微调每个建议，用户反馈持续优化 AI 表现。
4. **本地计算与在线智能结合**：基础任务管理、数据分析、日历视图等核心功能完全本地运行无需网络，AI 增强功能（自然语言解析、智能规划）通过 DeepSeek API 实现，需要网络连接但确保数据传输安全。
5. **可持续效率与节奏友好**：倡导温和提醒、用户可配置节奏与习惯建模，避免焦虑驱动的强制自动化，支持专注模式与工作生活平衡。
6. **开放扩展架构**：模块化设计自 Phase 0 起即为后续双智能体协同、团队协作、插件生态与社区扩展预留清晰的能力边界。

## Monitoring & Visibility

- **Dashboard Type**：Tauri 桌面应用内嵌的原生 React + shadcn/ui + Recharts 可视化仪表盘，支持暗/亮主题无缝切换与响应式布局。
- **Real-time Updates**：通过 Tauri IPC 事件通道、Zustand 状态管理与 Rust 后端计算实现毫秒级数据刷新，无需手动刷新页面即可查看最新分析结果。
- **Key Metrics Displayed**：
  - 任务完成统计：完成率、准时率、任务类型分布
  - 生产力评分：多维度评分拆解与历史趋势
  - 时间分配分析：按类型/优先级的时间投入热力图
  - 工作负载预测：未来 7/14/30 天负载预警与容量规划
  - 效率模式识别：高效时段识别与习惯养成追踪
  - AI 推荐采纳率：智能建议的接受度与反馈统计
- **Sharing Capabilities**：支持本地 Markdown/CSV/PNG 格式导出分析报告与任务清单，计划中支持加密数据包分享；所有导出操作完全在本地执行，确保数据安全。

## Future Vision

CogniCal 的长期愿景是成为业界领先的个人 AI 生产力平台：以先进的 AI 推理能力为核心，结合隐私保护的桌面架构，为个人工作与生活提供完整的计划—执行—复盘闭环体验。

### Potential Enhancements

- **双智能体完整实现**：完成任务规划智能体与生活助手智能体的协同机制，实现工作任务与生活节奏的智能平衡，提供全方位的个人管理支持。
- **深度个性化学习**：基于长期行为数据构建更精准的个人模型，提供更贴合个人工作风格的智能建议与习惯培养支持。
- **高级分析能力**：引入更精细的行为模式识别、异常检测、长期趋势预测与多维度对比分析，持续提升洞察价值。
- **团队协作功能**：扩展为多人工作空间、任务协作、评论互动与团队生产力仪表盘，支持权限管理与团队目标追踪。
- **插件与自动化市场**：提供开放的插件 API 与社区市场，支持用户自定义字段、扩展功能与分享工作流模板。
- **移动端支持**：开发配套的移动应用或 PWA，实现跨设备的任务同步与移动场景的快速操作。
