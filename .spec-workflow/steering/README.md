## 🎯 核心亮点

### 产品特色

1. **CoT 推理驱动** - 基于思维链的智能任务管理
2. **双智能体系统** - 任务规划 + 生活助手协作
3. **完全本地化** - 数据隐私至上,离线优先
4. **Tauri 2 框架** - 原生性能的跨平台桌面应用

### 技术架构

- **前端**: React 18 + TypeScript + Tailwind CSS + shadcn/ui
- **后端**: Tauri 2 + Rust + SQLite
- **AI 集成**: DeepSeek API + 自研 CoT 引擎 + MCTS 算法
- **双智能体**: 任务规划智能体 + 生活助手智能体

### 项目结构

- **清晰的分层架构**: UI 层 → 命令层 → 业务逻辑层 → 数据访问层
- **模块化设计**: 便于扩展和维护
- **完整的测试体系**: 单元测试 + 集成测试 + E2E 测试

## 💡 关键决策记录

### 技术选型理由

1. **Tauri 2**: 比 Electron 更轻量,原生性能更好
2. **Rust**: 内存安全,高性能,适合复杂算法
3. **React**: 成熟生态,丰富组件库
4. **SQLite**: 轻量级,完美支持本地存储
5. **DeepSeek API**: 国内可用,CoT 能力强

### 架构设计理由

1. **本地优先**: 隐私安全,离线可用
2. **双智能体**: 分工明确,各司其职
3. **CoT 推理**: 决策透明,可解释性强
4. **MCTS 算法**: 任务分解的最优解探索

## 🔒 隐私与安全承诺

- ✅ API 密钥安全存储(OS Keychain)
